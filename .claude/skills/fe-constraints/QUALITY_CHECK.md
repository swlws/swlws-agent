# 质量检查清单 (Quality Checklist)

在提交前端更改前，必须核对以下清单。

## 性能检查
- [ ] 是否对高频重绘路径上的组件使用了 `React.memo`？
- [ ] 传递给子组件的所有回调函数是否都使用了 `useCallback`？
- [ ] `useEffect` 的依赖数组是否完整且正确？
- [ ] 是否存在由于状态提升过高导致的非预期全页面重绘？

## 代码质量
- [ ] 是否移除了所有未使用的变量和导入？
- [ ] 是否遵循了具名导出的规范？
- [ ] 图标是否全部从 `@/fe/components/icons` 导入？
- [ ] 是否存在冗余的 CSS 类名或逻辑？

## 交互体验
- [ ] 图片渲染是否平滑（无闪烁，有加载态处理）？
- [ ] 按钮等交互元素是否有对应的禁用态 (disabled) 处理？
- [ ] 移动端适配是否正常？
